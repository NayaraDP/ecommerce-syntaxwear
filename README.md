# 👟 SyntaxWear - Tênis e Sneakers Online

O **SyntaxWear** é um projeto de e-commerce moderno e responsivo focado na venda de calçados (tênis e sneakers). O site foi desenvolvido com foco em uma experiência de usuário limpa, utilizando uma arquitetura de CSS baseada em componentes para garantir manutenibilidade e escalabilidade.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5**: Estrutura semântica para melhor SEO e acessibilidade.
- **CSS3**: Estilização avançada com o uso de **Variáveis CSS** (Custom Properties) e organização por **Componentes**.
- **Google Fonts**: Utilização das fontes *Ubuntu* para um design moderno.
- **Design Responsivo**: Adaptado para dispositivos móveis e desktops.

---

## ✨ Funcionalidades

- [x] **Cabeçalho Responsivo**: Menu hamburguer funcional para dispositivos móveis.
- [x] **Seção Hero**: Banner principal impactante com chamadas para ação (CTA).
- [x] **Categorias de Produtos**: Filtros visuais para estilos Casual, Esporte, Moderno e Futurista.
- [x] **Grade de Produtos (Grid)**: Layout dinâmico para exibição de destaques e novos modelos.
- [x] **Newsletter**: Formulário para captura de e-mails integrada ao rodapé.
- [x] **Rodapé Completo**: Links de navegação organizados e integração com redes sociais (Instagram, WhatsApp, TikTok, Facebook).

---

## 📁 Estrutura do Projeto

A organização dos arquivos segue o padrão de separação de responsabilidades:

```text
C:.
├── index.html              # Página principal do site
├── assets/                 # Recursos visuais
│   ├── images/
│   │   ├── banners/        # Imagens de hero e banners
│   │   ├── icons/          # Ícones de interface (SVG/PNG)
│   │   ├── logo/           # Logotipo da marca
│   │   └── products/       # Fotos dos produtos
└── css/                    # Estilização
    ├── base.css            # Estilos globais e tipografia
    ├── reset.css           # Reset de estilos padrão do navegador
    ├── variables.css       # Definição de cores e fontes (Variáveis)
    └── components/         # Estilos específicos de cada seção
        ├── footer.css
        ├── header.css
        ├── hero.css
        ├── product-category.css
        └── product-grid.css
```

## 🛠️ Como Executar o Projeto

Como o projeto é estático (HTML e CSS), não há necessidade de compiladores ou servidores complexos:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/ecommerce-syntaxwear.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd ecommerce-syntaxwear
   ```
3. Abra o arquivo `index.html` em qualquer navegador ou utilize a extensão **Live Server** do VS Code para uma melhor experiência de desenvolvimento.

## 🎨 Design e Estilo

- **Tipografia**: A fonte principal utilizada é a **Ubuntu**, que confere legibilidade e um ar tecnológico.
- **Componentização**: Cada parte da interface possui seu próprio arquivo CSS, facilitando alterações futuras sem afetar outras seções.
